# 🏦 Simulador de Conta Bancária em Java

Este projeto é uma simulação simples de um programa de **conta bancária**, desenvolvido com o objetivo de praticar conceitos de **Programação Orientada a Objetos (POO)** em **Java**.

---

## 🎯 Objetivo

O foco do projeto foi aplicar os seguintes conceitos que estou aprendendo no estudo de POO:

- 🧱 **Construtores**
- 🔒 **Encapsulamento**
- 🔁 **Sobrecarga de métodos**

---

## 💻 Tecnologias Utilizadas

- **Linguagem:** Java  
- **IDE:** IntelliJ IDEA 2024.3.5  
- **Paradigma:** Programação Orientada a Objetos

---

## 🚀 Funcionalidades

- Criar uma nova conta com ou sem depósito inicial  
- Exibir informações da conta formatadas  
- Realizar **depósitos** e **saques** com atualizações imediatas no saldo  
- Utilização de **métodos sobrecarregados** para diferentes formas de inicialização da conta  

---

## ▶️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/PedroVella/ex_conta_bancaria.git

2. Abra o projeto no IntelliJ IDEA 2024.3.5

3. Execute o arquivo principal (Conta.java)
